## Zola Theme Statistics - [click here for wider page](https://github.com/Jieiku/zola-themes/blob/main/README.md)

This list gives a quick overview of the different themes. I hope new developers can find it useful when they are deciding which themes to first use as an example to learn from. This list is not yet automated, so it will become outdated over time. The plan is to eventually automate this page if I can ever find the time. For now I will post the last updated date. Feel free to open an issue report if an entry has big changes and needs updated.

Last Updated: June 22 2022

- KB: Kilobytes Blog Index (lower is better)
- GL: Google Lighthouse Average Score (higher is better)
- MO: Mozilla Observatory Score (higher is better)
- S: Search is implemented.

Themes with working Search implemented are at the top, sorted by Lighthouse and Observatory rating, the rest are sorted by star rating.

Themes without a demo, that wont build, or that the repository is gone are at the bottom of the list.
